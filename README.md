# Build In Public Hub

## Overview
A "Proof of Work" dashboard for indie hackers that aggregates stats from GitHub, X, and Product Hunt.

## Project Structure
- `web/`: Next.js Frontend & API (Next.js 15, Tailwind CSS)
- `scripts/`: Data ingestion scripts (Coming Soon)

## Getting Started

### Prerequisites
- Node.js & npm

### Installation
1. Navigate to the web directory:
   ```bash
   cd web
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Features (Planned)
- **Developer Profiles**: GitHub/X/Product Hunt stats.
- **Hustle Leaderboard**: Real-time ranking.
